Grundbegriffe

📝 Formale Grammatik

G = (N, T, P, S)
N = Nichtterminale (Variablen)
T = Terminale (Alphabet)
P = Produktionsregeln
S = Startsymbol (S ∈ N)

🔄 Ableitung

Ersetzung von Nichtterminalen durch rechte Seite einer Regel:
S ⇒ aSb ⇒ aaSbb ⇒ aabb
Jeder Schritt wendet genau eine Regel an.

🗂️ Sprache L(G)

Die Menge aller Wörter aus Terminalen, die aus S ableitbar sind:
L(G) = {w ∈ T* | S ⇒* w}

💡 Hierarchie

Jede reguläre Sprache ist auch kontextfrei, jede kontextfreie auch kontextsensitiv, usw. Typ 3 ⊂ Typ 2 ⊂ Typ 1 ⊂ Typ 0.

🎯 Prüfungstipp

Erkenne den Typ an der Regelform: Rechts nur ein Terminal + optional ein Nichtterminal = Typ 3. Links nur ein Nichtterminal = Typ 2.

🧩 Automat-Zuordnung

Typ 3 → DEA/NEA, Typ 2 → Kellerautomat (PDA), Typ 1 → LBA, Typ 0 → Turingmaschine. Stärkerer Automat = mächtigere Sprachen.
