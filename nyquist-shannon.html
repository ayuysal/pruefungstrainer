Die drei Kernformeln

Nyquist-Datenrate

NYQUIST RATE · rauschfreier Kanal
C = 2 · B · log₂(M)
C = max. Datenrate (Bit/s)
B = Bandbreite des Kanals (Hz)
M = Anzahl Signalstufen

Gibt die maximale Datenrate auf einem rauschfreien Kanal an. Mehr Signalstufen → mehr Bit pro Symbol.
Einheit: Bit/s

Shannon-Hartley

CHANNEL CAPACITY · verrauschter Kanal
C = B · log₂(1 + S/N)
C = Kanalkapazität (Bit/s)
B = Bandbreite (Hz)
S/N = Signal-Rausch-Verhältnis (linear)

Die theoretische Obergrenze der fehlerfreien Übertragung auf einem verrauschten Kanal. Gilt als das fundamentalste Theorem der Informationstheorie.
Einheit: Bit/s

Abtasttheorem

SAMPLING THEOREM · Nyquist-Shannon
fₛ ≥ 2 · f_max
fₛ = Abtastfrequenz (Hz)
f_max = höchste Signalfrequenz (Hz)

Ein Signal muss mit mindestens der doppelten seiner höchsten Frequenz abgetastet werden, um es verlustfrei rekonstruieren zu können. Sonst: Aliasing.
fₛ(min) = 2 · f_max

🔗 Zusammenspiel der Formeln

Nyquist sagt: So viele Bit/s passen maximal auf den Kanal (ohne Rauschen). Shannon-Hartley sagt: So viele Bit/s gehen maximal TROTZ Rauschen. Abtasttheorem sagt: So oft muss ich mindestens abtasten, um nichts zu verlieren. → Die tatsächliche Datenrate ist immer ≤ min(Nyquist, Shannon). → Nyquist begrenzt durch Bandbreite + Signalstufen. → Shannon begrenzt durch Bandbreite + Rauschen. → Das Abtasttheorem ist die Voraussetzung für korrekte Digitalisierung.

🎯 Prüfungstipp

Aufpassen: SNR in dB muss zuerst in linear umgerechnet werden! SNR_lin = 10^(SNR_dB / 10). Häufiger Fehler in Klausuren.

💡 Merkhilfe

Nyquist = rauschfrei (idealer Kanal). Shannon = realistisch (mit Rauschen). Shannon ist immer die absolute Obergrenze.

🎵 Praxisbeispiel

Audio-CD: f_max = 22'050 Hz Abtastrate (> 2 × 20'000 Hz Hörgrenze), 16 Bit Tiefe, 2 Kanäle → Datenrate = 44'100 × 16 × 2 = 1'411'200 Bit/s.
