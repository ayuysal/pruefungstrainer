# Interaktive Lernplattform – ITI & TGI Prüfungsvorbereitung

Interaktive HTML-Trainer zur Vorbereitung auf zwei Prüfungen:
- **Informationstechnologie (IT-Infrastruktur)** – Hefte ITI21-26
- **Theoretische Grundlagen der Informatik** – Hefte GDI01, TGI01, TGI02, TGI06, TGI07

## Features

- 30 interaktive Standalone-Trainer (HTML, kein Server nötig)
- Dark Theme mit modernem Design
- Lernfortschritt wird lokal im Browser gespeichert
- Glossar-System mit 3-Phasen-Lernen (Kennenlernen → Überprüfen → Beweisen)
- Gedächtnispalast (Loci-Methode) mit vorgenerierten Merk-Geschichten
- Freies Erinnern mit Scaffold/Ghost-Text-Unterstützung
- Lern-Navigator mit Stärken/Schwächen-Analyse
- Mock-Exam Simulator
